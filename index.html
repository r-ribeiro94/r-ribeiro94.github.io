Todo List Application

This a simple todo list application that uses AngularJs. It reads a JSON file with pre-loaded tasks and once the user adds another task it gets appened to the list but not to the JSON file. The user must input a title for the task and a description if neither is inputed they will be prompted to input both fields.

The remove checked button will remove all the checked tasks and stored it in a list of completed tasks. The completed tasks button will show/hide the completed tasks.

  • Client: Github

  • Date: March 2016

  • Service: Web Applications (CPS 630)

Drag and Drop Location Application

This application takes advantage of the HTML5 geolocation and drag and drop feature. It reads a text file that has one coordinate in longitude and latitude format (ex: 47.000,-74.000). Once the text file has been dropped into the box, it will show both the location of the user and that of the text file.

The 'Get Harvestine' button uses Web Worker to calculate the distance between the user's location and that of the text file, and the 'Stop Worker' button stops the Web Worker which doesn't allow the user to use the 'Get Harvestine' button, because the Web Worker has been dismissed.

  • Client: Github

  • Date: February 2016

  • Service: Web Applications (CPS 630)
