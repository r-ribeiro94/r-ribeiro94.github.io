Hello,

As of Spring 2016, I am a graduate of Ryerson University with a Bachelors of Science in Computer Science. During my time at Ryerson, I became very interested in any web related topics either as a front-end or back-end contributor.

Starting out only as an assignment during my 4th year, I helped develop a scheduling application along with two other colleagues of the Computer Science program. My colleagues and I were encouraged to continue working on this application from our professor as we have currently taken time out of our schedules to do so. The tasks are distributed equally as my contribution to this project is related to the user experience and front-end development of the application.

During my participation in this project, I have been able to enhance my skills in HTML and CSS, as well as furthering my knowledge of APIs and frameworks such as Google APIs, Bootsrap, Fullcalendar and OAuth.

Outside of academics, my interests lie in playing video games and participating in sports such as soccer. Through these activities I have become a more team focused, result oriented and reliable person. These traits have allowed me to solve everyday problems and thrive in a team environment with an improved focus in achieving goals and going beyond expectations.

Todo List Application

This a simple todo list application that uses AngularJs. It reads a JSON file with pre-loaded tasks and once the user adds another task it gets appened to the list but not to the JSON file. The user must input a title for the task and a description if neither is inputed they will be prompted to input both fields.

The remove checked button will remove all the checked tasks and stored it in a list of completed tasks. The completed tasks button will show/hide the completed tasks.

  • Client: Github

  • Date: January 2016

  • Service: Web Applications (CPS 630)
